How to properly measure box sash windows

How to Measure a Window - Structural Opening

Accurate measurements are crucial for a perfect window fit. Follow these steps carefully to ensure your new windows will fit perfectly. Always measure in millimeters and take multiple measurements at different points, recording the smallest dimension.

  1. Measure Width (Brick to Brick)

    Measure the width of the opening at the top, middle, and bottom. Record the smallest dimension obtained. This ensures your window will fit even if the opening varies slightly.

  2. Measure Height

    Measure the height of the opening on the left side, center, and right. Record the smallest dimension. This ensures proper vertical fit throughout the opening.

  3. Measure Depth

    Measure the depth from the outer brick to the inner surface, noting any obstacles such as pipes, wires, or protrusions that might affect installation.

  4. Check for Square Corners

    Check if the opening is square by measuring diagonally from corner to corner. Both diagonal measurements should be equal. If they differ significantly, note this for the installer.

  5. Measure Structural Opening (Brick to Brick)

    Measure the rough opening from brick to brick for both width and height to obtain the structural dimensions. This is the actual opening size in the wall structure.

  6. Measure Frame Dimensions (if applicable)

    Finally, measure the internal dimension of the existing window frame (if installed). This helps determine if a replacement or new installation is needed.

Required Tools

  • Retractable tape measure
  • Spirit level
  • Pencil
  • Notepad
  • Calculator (optional)
